# Doremiwav

A simple wav file audio generator by Ruby.

## Dependencies

* [jstrait/wavefile](https://github.com/jstrait/wavefile)

## Installation

    $ gem install doremiwav

## Usage

    $ doremiwav generate "AAAA AA      AAAAAAAA"
